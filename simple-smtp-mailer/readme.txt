TechWeirdo’s professional SMTP plugin with custom SMTP server settings for any provider.

== Description ==
TechWeirdo’s Simple SMTP Mailer is a professional WordPress plugin that enables you to configure custom SMTP server settings for sending emails from your WordPress site. This plugin allows seamless integration with any SMTP provider, ensuring reliable email delivery with options for encryption, authentication, and debug logging.

**Key Features:**
- Custom SMTP host, port, and encryption settings (SSL/TLS).
- Configurable "From" email and name.
- Test email functionality to verify SMTP settings.
- Debug mode for troubleshooting email issues.

For detailed documentation, visit [TechWeirdo.net](https://techweirdo.net).

== Installation ==
1. Download the plugin from the [GitHub repository](https://github.com/drshounak/wordpress-plugins/simple-smtp-mailer).
2. Upload the `simple-smtp-mailer` folder to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Navigate to **Settings > SMTP Mailer** to configure your SMTP settings.

== Usage ==
1. Go to **Settings > SMTP Mailer** in your WordPress admin dashboard.
2. Enter your SMTP server details (host, port, encryption, username, password).
3. Configure the "From" email and name for outgoing emails.
4. Enable debug mode if troubleshooting is needed.
5. Use the "Send Test Email" feature to verify your configuration.
6. Save your settings and start sending emails reliably.
